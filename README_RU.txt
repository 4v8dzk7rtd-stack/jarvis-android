JARVIS Android v0.1

Функции:
- кнопка микрофона;
- системное распознавание русской речи;
- отправка текста на http://192.168.1.164:8765/chat;
- токен jarvis-local;
- получение и отображение ответа;
- системный синтез речи Android.

Перед использованием:
1. Телефон и компьютер должны быть подключены к одной Wi-Fi сети.
2. Ollama и JARVIS Server должны быть запущены.
3. При первом старте разрешить приложению доступ к микрофону.

Это исходный проект Android. Для сборки требуется Android SDK/Gradle или GitHub Actions.
